PlusPlugins Overview
</h1>

**Welcome to PlusPlugins!**

PlusPlugins is a set of Flutter plugins that is developed based on existing Flutter plugins with extra functionalities, platforms support, and better maintenance.

:::caution
PlusPlugins is currently a work in-progress, as we aim towards a stable release. Plugins & documentation may be incomplete
or missing functionality. See the [roadmap](https://github.com/fluttercommunity/plus_plugins/issues/12) to learn more.
:::

```yaml {4} title="pubspec.yaml"
dependencies:
  flutter:
    sdk: flutter
  batter_plus: "{{ plugins.batter_plus }}"
```

Install the plugin by running the following command from the project root:

```bash
$ flutter pub get
```
